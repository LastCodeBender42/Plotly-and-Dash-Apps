## Cloning the repository: 
In the repository navigate to the dashboard-with-network directory. Find and select the green 'Code' button as shown below and then copy the link in the drop down menu. 
<br></br>
<img src="clone_the_repo.png" alt="Example Image">
<br></br>
Next, open a terminal on your Desktop and type `git clone git@github.com:LastCodeBender42/PlotlyDashApps.git`. This automatically creates a clone of the folder on your Desktop with the requisite files to deploy the app.
